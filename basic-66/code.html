2. El Modelo en MRJob

Este modelo se divide en 3 partes. Primeramente, la función PREDICT() que genera las probabilidades de 0 o 1 dependiendo de un umbral de probablidades. El parámetro "p_pred" es la probabilidad estimada por la funcion logit() de que la observación perteneza a la categoría de "mal pagador", que correspondería al valor 1 de la variable dependiente. En segundo lugar, la función LOGIT() calcula las predicciones a partir de los pesos iniciales, el parámetro "pesos" es la suma se los bias y los betas almacenada en una lista y el parámetro "variables" es la lista de las variables de entrenamiento. Por último, la función ACCURACY() genera la métrica de accuracy para evaluar el performance del modelo.

              def predict(p_pred,threshold=0.5):
                  return np.array([1 if y>threshold else 0 for y in p_pred])


              def accuracy(y_pred,y):
                  error_prediccion = [ y_pred[i]==y[i] for i in range(0,len(y_pred))]
                  return sum(error_prediccion)/len(y_pred)

              def logit(variables,pesos):
                  variables=np.array(variables)
                  intercepto = np.ones((variables.shape[0], 1))
                  variables =np.hstack((intercepto, variables))
                  pesos = np.array(pesos)

                  regresion=[sum(z) for z in pesos.T * variables]
                  output = [1/(1+np.exp(-reg)) for reg in regresion]
                  return np.array(output)


              #FUNCIONES PRINCIPALES

                          #VALOR OBJETIVO A PREDECIR

                      # gradiente del error en funcion del bias
